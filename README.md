# Fault Registration System v2.1

نظام تسجيل الأعطال العام - إصدار 2.1

## نظرة عامة

هذا النظام هو تطبيق ويب مبني باستخدام Flask و SQLite لإدارة وتوثيق أعطال الأجهزة في أي مؤسسة. يوفر النظام واجهة سهلة الاستخدام لتسجيل الأعطال، تتبعها، وإنشاء تقارير شاملة.

## الميزات الرئيسية

### إدارة الأعطال
- تسجيل أعطال جديدة مع تفاصيل شاملة
- تتبع حالة العطل (مفتوح، قيد الإصلاح، مغلق)
- حساب تلقائي لوقت الإصلاح (MTTR)
- تعيين الموظفين المسؤولين عن الإصلاح

### إدارة الأجهزة
- بطاقات تفصيلية للأجهزة الرئيسية والثانوية
- تايم لاين لكل جهاز يسجل الأحداث والصيانة
- إحصائيات شاملة لكل جهاز

### التقارير والتحليلات
- تقارير KPI (عدد الأعطال، MTTR، معدل الإغلاق)
- تحليل الأعطال حسب المشروع والجهاز
- أداء الموظفين
- الأجهزة الأكثر تعطلاً
- تصدير التقارير إلى PDF بتنسيق A4

### إدارة قاعدة البيانات
- إدارة المشاريع والأجهزة والموظفين
- إضافة وتعديل البيانات الأساسية

## المتطلبات

- Python 3.7+
- Flask
- SQLite3 (مدمج مع Python)

## التثبيت والتشغيل

1. قم بتثبيت المتطلبات:
   ```bash
   pip install flask
   ```

2. قم بتشغيل التطبيق:
   ```bash
   python app.py
   ```

3. افتح المتصفح واذهب إلى: `http://localhost:5000`

## قاعدة البيانات

يستخدم النظام قاعدة بيانات SQLite مع الجداول التالية:

- `projects`: المشاريع
- `MainDevices`: الأجهزة الرئيسية
- `SubDevices`: الأجهزة الثانوية
- `Employees`: الموظفين
- `problems`: الأعطال
- `DeviceTimeline`: سجل الأحداث للأجهزة

## الاستخدام

### تسجيل عطل جديد
1. اذهب إلى "إضافة عطل"
2. اختر المشروع والجهاز
3. أدخل وصف العطل
4. حدد المسؤول عن الإصلاح

### عرض الأعطال
- استخدم صفحة "عرض الأعطال" للبحث والفلترة
- انقر على اسم الجهاز لعرض بطاقته التفصيلية

### التقارير
- اذهب إلى صفحة "التقارير"
- اختر الفترة والمشروع
- عرض التحليلات المختلفة
- تصدير إلى PDF
